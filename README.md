# GitHub Profile Application

A mobile application built with Kotlin that showcases GitHub user profiles and repositories.

## Features

- View GitHub user profiles
- Browse user repositories
- Clean and intuitive user interface
- Offline support

## Tech Stack

- **Language**: Kotlin, Java
- **Build Tool**: Gradle
- **Platform**: Android

## Getting Started

### Prerequisites

- Android Studio Otter 2025.2.1 or later
- JDK 8 or higher
- Android SDK

### Installation

1. Clone the repository
2. Open the project in Android Studio
3. Sync Gradle files
4. Run the application

## Usage

Launch the app and enter a GitHub username to get your latest insights on your GitHub profile and repositories.
